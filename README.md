# Focus Mode Extension for VScode

**Автор:** Ахмед-Алиев Руслан Эхтибарович M3115

## Описание
**Focus Mode** - расширение для VScode, которое помогает концентрироваться на работе. Плагин позволяет запускать таймер, скрывать отвлекающие элементы интерфейса, ввести статистику времени работы.

---
## Возможности плагина

- **Таймер фокус-сессии**: пользователь вводит длительность сессии в минутах; таймер отображается в строке состояния.  
- **Скрытие отвлекающих элементов**: закрытие боковой панели и панели терминала.  
- **Статистика сессий**: сохраняется дата, длительность и имя файла, на котором работал пользователь.  
- **Отчёты**: просмотр всех завершённых сессий через Webview.  
- **Интерактивность**: запуск/остановка сессий через командную палитру, горячую клавишу или клик по строке состояния.

---

## Установка

1. Скачайте файл плагина `FocusMode-0.0.1.vsix`.  
2. Откройте VS Code.  
3. Перейдите в **Extensions** (`Cmd+Shift+X` на macOS или `Ctrl+Shift+X` на Windows).  
4. Нажмите на три точки `...` → **Install from VSIX…**.  
5. Выберите файл `.vsix` и установите плагин.  

---

## Использование
### Запуск сессии
#### Через командную палитру:
1. Нажмите `Cmd+Shift+P` (MacOS) или `Ctrl+Shift+P` (Windows/Linux)
2. Введите `Toggle Focus Mode` и нажмите enter
3. Введите длительность в минутах

### Просмотр отчёта
#### Через командную палитру:
1. Нажмите `Cmd+Shift+P` (MacOS) или `Ctrl+Shift+P` (Windows/Linux)
2. Введите `Show Report Focus Mode` и нажмите enter
3. Откроется Webview с историей всех сессий
